# Security Policy

## Reporting a vulnerability

Please report suspected vulnerabilities through GitHub Security Advisories:

- https://github.com/joehoot/gantryland/security/advisories/new

If advisory reporting is unavailable, open a private report by contacting the maintainers through repository owners on GitHub and include "security" in the subject.

When reporting, include:

- Affected package(s) and version(s)
- Reproduction steps or proof of concept
- Expected impact and exploitability notes
- Any suggested remediation

## Response targets

- Initial acknowledgement: within 3 business days
- Triage and severity assessment: within 7 business days
- Remediation timeline: communicated after triage

## Disclosure policy

- Please do not open public issues for unpatched vulnerabilities.
- We aim for coordinated disclosure after a fix is available.

## Supported versions

Security fixes are prioritized for the latest published minor line.
