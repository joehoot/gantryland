# Support

## Getting help

- Usage questions and troubleshooting: open a GitHub issue.
- Bugs and regressions: open a GitHub issue with reproduction steps.
- Security concerns: follow `SECURITY.md`.

## Scope

Support is best-effort for:

- Latest published package versions
- Current `main` branch behavior

## What to include in requests

- Package name(s) and version(s)
- Runtime and tooling versions (Node.js, TypeScript)
- Minimal reproduction code
- Observed behavior and expected behavior
